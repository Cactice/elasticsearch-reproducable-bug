# elasticsearch-reproducable-bug
## Reproduce procedure
1. Start elastic search
`docker-compose up`
2. Install dependency
`yarn`
3. Run dump command
`yarn dump`

## example.json
example.json is the first 20 line downloaded of wikimedia's elasticsearch dump
